# Ducumentação sobre o Projeto Jarvis

---
Cada arquivo contém uma análise técnica profunda de um ou mais módulos do seu projeto, explicando sua função, as bibliotecas utilizadas, as partes mais importantes do código e como eles se conectam para formar o sistema completo do Jarvis.

Esta documentação servirá como uma excelente referência técnica para o desenvolvimento contínuo do projeto.

1. [`app.py`](app_analysis.md): Este arquivo é o ponto de entrada para as interfaces CLI e Web.
2. [`gui_app.py`](gui_app_analysis.md): Este arquivo é o ponto de entrada para a interface gráfica.
3. [`jarvis.py`](jarvis_analysis.md): Este é o arquivo mais importante, o cérebro do projeto.
4. [`nlp_client`, `rag`, `sentiment`](ai_modules_analysis.md): Análise dos módulos que lidam diretamente com os modelos de IA.
5. [`memory.py`, `task_manager.py`](memory_systems_analysis.md): Análise dos dois tipos de memória do Jarvis.
6. [**Módulos de Áudio e Utilitários**](utility_modules_analysis.md): Análise dos arquivos restantes.
